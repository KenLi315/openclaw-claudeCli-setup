# 期權客戶計息與出金手續費政策優化方案

- **建立日期**: 2026-03-27
- **提案人**: Ken
- **狀態**: 構思中 (Idea)

## 1. 核心想法 (Core Concept)
針對期權客戶存款與提款政策進行「推拉」優化，以達到留存資金、降低成本、防範風險的目標。

## 2. 存款誘因 (Incentive: The Pull)
- **方案**: 計算存款利息予期權客戶。
- **息率**: 基準利率 (Benchmark) 減 100 個基點 (100 bps)。
- **效益**: 增加客戶將資金留在公司的意願，減少頻繁出金回銀行的行為。

## 3. 提款限制 (Penalty: The Push)
- **方案**: 對每次出金收取手續費。
- **執行方式**: 每次出金收取固定手續費或設置最低收費 (Minimum Charge)。
- **效益**: 提高客戶頻繁提款的行政成本，進一步鞏固資金留存效果。

## 4. 戰略效益 (Strategic Benefits)
- **降低資金成本**: 將客戶存款轉化為低成本資金來源。
- **防範違約風險**: 減少客戶欠款 (爭錢) 與違約 (走數) 的機會。
- **運營效率**: 降低行政處理頻繁出金的負擔。

## 5. 後續跟進 (Next Steps)
- 模擬分析當前 HIBOR/SOFR 扣減 100bps 後的競爭力。
- 制定具體的出金手續費收費標準。
- 起草正式的政策通知。
